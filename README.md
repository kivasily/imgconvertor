# imgconvertor
<h1>Simple Local IMG converter</h1>
<h3>About the project</h3>
<p>Look! I've made a simple image converter. It works both via a link (https://kivasily.github.io/imgconvertor/) and locally. To download it to your device, visit the repository page (https://github.com/kivasily/imgconvertor/), download the main file (index.html), and open it with any browser. All conversions happen directly on your device in your browser, making this project 100% confidential and secure. All the source code, which you can download and use locally, is also used on my website (via the GitHub page). You don't have to worry about speed—if you have an average device, you don't have to worry about it at all! (By "old devices," we mean devices with less than 8 GB of RAM and an old CPU (e.g., Intel Pentium or Intel Celeron)). The converter can convert images from and to these formats: <b>PNG, JPG, JPEG, WEBP, AVIF, HEIС, and SVG</b>. There may be some problems with the SVG format due to the specifics of this format.</p>
